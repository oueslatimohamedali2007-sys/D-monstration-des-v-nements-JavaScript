Résumé JavaScript – Appel des fonctions

1. JavaScript

JavaScript est un langage de programmation pour le web. Il rend les pages HTML interactives après leur chargement.

2. Où écrire le JavaScript ?

  • Dans la balise <head>
  • Dans la balise <body>
  • Dans un fichier externe .js
<script src="script.js"></script>

3. Déclaration d’une fonction

function direBonjour() {
  alert("Bonjour");
}

4. Types d’appel des fonctions

Type Rôle Exemple
onclick Clic souris <button onclick="f()">
onload Chargement page <body onload="f()">
onsubmit Envoi formulaire <form onsubmit="f()">
onmouseover Survol souris <div onmouseover="f()">
onchange Changement valeur <input onchange="f()">
addEventListener Bonne pratique element.addEventListener("click", f)
setTimeout Appel différé setTimeout(f, 1000)
setInterval Appel répété setInterval(f, 1000)
clicker ici pour les tester

5. Accès aux éléments HTML

document.getElementById("id").value
document.getElementById("id").checked
document.getElementById("id").selectedIndex

Phrase clé pour l’examen

Une fonction JavaScript peut être appelée par un événement HTML, par JavaScript directement ou automatiquement.
